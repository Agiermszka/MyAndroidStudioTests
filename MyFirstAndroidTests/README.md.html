1    # InstaList comm 
2    
3    Communication project for [InstaList](https://github.com/InstaList/instalist-android) and it's  
4    [Server](https://github.com/InstaList/instalist-server).  
5    
6    Currently, InstaList uses a REST-API accepting and creating JSON for transmitting data. The messages 
7    can get created and parsed in a easy via [Jackson](https://github.com/FasterXML/jackson) and it's  
8    `databind` module. 
9    
10   Classes that can be used for this purpose are located in package  
11   `org.noorganization.instalist.comm.message`. 
12   
13   ## Usage 
14   
15   Use these classes as a library, for example via gradle. This library is not available in the  
16   central repository. But it can be included with a master gradle-project like this: 
17   
18       // gradle.settings 
19       include ':instalist-comm' 
20   
21       // build.gradle 
22       dependencies { 
23           compile(':instalist-comm') 
24           // ... 
25       } 
26   
27   This works if your master is one directory above this project. 
28   
29   You can create IntelliJ-IDEA compatible project files with: 
30   
31       ./gradlew idea
